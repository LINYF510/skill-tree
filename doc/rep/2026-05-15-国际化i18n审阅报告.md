### ✅ 任务 9：ErrorStateManager 国际化 — PASS（亮点）

**这是本次实现中最彻底的重构**：

| 组件 | 改造 |
|------|------|
| UiError | String → @StringRes Int + messageArgs |
| ErrorAction | String label → @StringRes Int labelResId |
| SnackbarEvent | String message → @StringRes Int messageResId + messageArgs |
| DialogEvent | String title/message → @StringRes Int + messageArgs |
| ErrorStateManager.showSnackbar | (String) → (Int, Array<Any>) |
| ErrorStateManager.showErrorDialog | (String, String) → (Int, Int, Array<Any>) |
| ErrorStateManager.mapDomainException | 返回 UiError 使用 R.string.error_xxx |
| ErrorStateManager.mapDataException | 返回 UiError 使用 R.string.error_xxx |
| MainActivity Snackbar/Dialog | event.message → context.getString(event.messageResId, *event.messageArgs) |

**亮点**: UiError 正确重写了 equals/hashCode（处理 Array 的 contentEquals/contentHashCode）。
