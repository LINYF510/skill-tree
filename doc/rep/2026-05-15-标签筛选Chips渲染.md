# 2026-05-15-标签筛选Chips渲染

## 实现概述

修复 SearchBar 组件接收了 `allTags`/`selectedTags`/`onTagClick` 参数但未渲染标签筛选 Chips 的问题。同时修复 `allTags` 数据源为空的问题，将其从本地空列表改为通过 ViewModel 从 `GetAllTagsUseCase` 获取。

### 主要变更

1. **ViewModel 层**：在 `SkillTreeViewModel` 中注入 `GetAllTagsUseCase`，新增 `allTags` 状态流，在 `init` 中调用 `loadAllTags()` 加载所有标签
2. **Screen 层**：移除 `SkillTreeScreen` 中本地 `allTags` 空列表和空 `LaunchedEffect`，改为从 ViewModel 收集 `allTags`
3. **SearchBar 组件**：在搜索历史区域下方新增标签筛选区域，使用 `FilterChip` 横向滚动展示所有标签，每个标签显示颜色点和名称，选中标签用 PrimaryBlue 高亮
4. **辅助函数**：新增 `parseColor()` 函数，将十六进制颜色字符串解析为 Compose Color

## 关键代码片段

### ViewModel 新增 allTags 状态流

```kotlin
private val _allTags = MutableStateFlow<List<TagEntity>>(emptyList())
val allTags: StateFlow<List<TagEntity>> = _allTags.asStateFlow()

private fun loadAllTags() {
    viewModelScope.launch {
        getAllTagsUseCase()
            .catch { e ->
                _uiState.update {
                    it.copy(errorMessage = e.message ?: "加载标签失败")
                }
            }
            .collect { tags ->
                _allTags.value = tags
            }
    }
}
```

### SearchBar 标签筛选区域

```kotlin
if (allTags.isNotEmpty()) {
    Column(
        modifier = Modifier
            .fillMaxWidth()
            .padding(horizontal = 16.dp, vertical = 8.dp)
    ) {
        Text(text = "标签筛选", color = TextSecondary, fontSize = 13.sp)
        Spacer(modifier = Modifier.height(8.dp))
        Row(
            modifier = Modifier
                .fillMaxWidth()
                .horizontalScroll(rememberScrollState()),
            horizontalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            allTags.forEach { tag ->
                val isSelected = selectedTags.contains(tag.id)
                val tagColor = tag.color?.let { parseColor(it) } ?: TextSecondary
                FilterChip(
                    selected = isSelected,
                    onClick = { onTagClick(tag.id) },
                    label = {
                        Row(
                            verticalAlignment = Alignment.CenterVertically,
                            horizontalArrangement = Arrangement.spacedBy(4.dp)
                        ) {
                            Canvas(modifier = Modifier.width(8.dp).height(8.dp)) {
                                drawCircle(color = tagColor)
                            }
                            Text(
                                text = tag.name,
                                color = if (isSelected) TextPrimary else TextSecondary,
                                fontSize = 13.sp
                            )
                        }
                    },
                    colors = FilterChipDefaults.filterChipColors(
                        selectedContainerColor = PrimaryBlue.copy(alpha = 0.2f),
                        containerColor = BackgroundDark
                    ),
                    border = FilterChipDefaults.filterChipBorder(
                        borderColor = if (isSelected) PrimaryBlue else TextSecondary.copy(alpha = 0.3f),
                        selectedBorderColor = PrimaryBlue,
                        enabled = true,
                        selected = isSelected
                    )
                )
            }
        }
    }
}
```

### parseColor 辅助函数

```kotlin
private fun parseColor(hex: String): Color {
    return try {
        val cleanHex = hex.removePrefix("#")
        val colorValue = when (cleanHex.length) {
            6 -> "FF$cleanHex".toLong(16)
            8 -> cleanHex.toLong(16)
            else -> return TextSecondary
        }
        Color(colorValue)
    } catch (_: IllegalArgumentException) {
        TextSecondary
    }
}
```
